Week 2 Blog Post

Moulthrop’s idea on hypertext as a radical shift on how we interact with information have played out, but they also have unexpected consequences. He talks about how hypertext improves navigation and access to relevant information, even though he does talk about whether the ease is always beneficial. Hypertext helps users navigate through the web using links between relative topics and allows a linear navigation though the web. Hypertext user to jump between sources quickly and effectively and discovering more connections between pages, this also allows the user to have a personalised information experience instead of relying on fixed and one way narratives. The negative impact of hyperlinks may be that users may not engage fully into certain sources by jumping endlessly through the links and losing the original context.

Below is my folder structure and wireframe, this is one of the examples of how hypertext can influence navigation through certain sources and pieces of information. The wireframe showcases the freedom and easy access to certain information that will be provided by my webpage. The lay out shows how users can navigate the whole website from a click of a mouse.
The folder structure consists of four folders that represent 4 of my current planned pages and each one is going to have four folders in each folder, each one being Images, Styles, Scripts and HTML folders. Each files consists of these folders to store the files and make it easier to navigate through the files that are needed in the webpage.

My webpage design or wireframe was inspired by my experience with the World wide web. The websites and webpages that I have visited were easy to navigate and easy to use. These websites are designed to make the human or use experience easier to use and not the other way around where a use has to learn how to use the webpage and navigate through based on what information was presented to me, instead it was built so that I can navigate the web based on what I want to engage in. Sometime the world wide web can present false information and unnecessary information to the user in the form of an add, sometimes it will present a video and if a user watches the video for more than 10 or more seconds, the web thinks that the user wants content like that and ends up presenting more information that’s related to that one video.
